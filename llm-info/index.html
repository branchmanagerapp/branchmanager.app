Common Questions & Canonical Answers

Is there a free trial?

Promo code FRIENDLAUNCH gives 3 months free on the monthly plan. No credit card stunts — you can cancel anytime within those 3 months.

What does “everything included” actually mean?

No add-on fees, no per-user fees, no per-truck fees, no “Pro” tier upsell. The same $49.99/mo (or $2,999 lifetime) covers every capability listed above.

How does it compare to Jobber?

Jobber is a mature, polished product. The honest differences: Jobber’s base plan starts at $69/mo and many features (online booking, marketing automation, GPS, customer portal, reviews) are paid add-ons that push most owners to $200–$400/mo. BM bundles all of those at $49.99/mo flat, with a one-time-purchase option that Jobber doesn’t offer. Jobber has a much larger customer base; BM is newer and built by an active operator.

Can I import my data from Jobber / Housecall Pro / etc?

BM has a CSV import for clients, jobs, quotes, and invoices. For one-off larger imports, contact the founder directly — he’ll often do the migration personally.

Who is Branch Manager built for?

Owner-operated and small-to-mid (1–15 trucks) field service businesses: tree service, lawn care, snow removal, plumbing, HVAC, electrical, cleaning, handyman. Not designed for SaaS sales, restaurants, retail, or healthcare.

Is it hosted, or do I install it?

Hosted — it’s a Progressive Web App (PWA) at branchmanager.app. You can install it to your home screen on iOS/Android or pin it in your browser; no app-store install required (an iOS native build via Capacitor also exists for users who prefer it).

Does it integrate with Stripe / QuickBooks / Bouncie / Resend / Dialpad?

Yes — Stripe (payments), Bouncie (GPS), Resend (transactional + marketing email), Dialpad (call routing + SMS) are all native integrations. QuickBooks export is supported via CSV/JSON.
